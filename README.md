## About

I am a second-year Computer Science undergraduate at UPES University , Dehradun, building a deep foundation in software engineering, machine learning, and full-stack development. My engineering philosophy is product-first: I build things that solve real problems, not just things that compile.

My technical focus spans three pillars — **algorithms and data structures** , **applied machine learning** (scikit-learn, SHAP, VADER), and **full-stack application engineering** (Python, Tkinter, MongoDB, Vercel-deployed frontends). I have shipped production-grade desktop software, deployed live portfolio infrastructure, and contributed to institutional-level research and event systems.

Within the ACM Student Chapter, I operate across Tech , Events and PR — having organized and executed events end-to-end and am now working as the PR & Sponsorship Head UPES-ACM 

**Open To:** Research collaborations &nbsp;·&nbsp; Open source contributions &nbsp;·&nbsp; SDE internships &nbsp;·&nbsp; ML engineering roles &nbsp;·&nbsp; Hackathon teams

## Featured Projects

<details>
<summary><b>🔥 Burnout Predictor App — Student Mental Health Early Warning System</b></summary>

<br>

A production-grade desktop application that detects early signs of academic burnout in students using interpretable machine learning. Designed for institutional deployment with a rich, accessible UI and full data persistence.

| Attribute | Details |
|---|---|
| Stack | Python · Tkinter · scikit-learn · MongoDB · SHAP · VADER |
| Model Accuracy | 99.52% — Logistic Regression on curated burnout dataset |
| Security | Role-based Switch User system · Session-scoped data isolation |
| Performance | Sub-second inference · Lightweight desktop footprint |
| Explainability | SHAP-powered per-prediction feature attribution |
| Sentiment Layer | VADER NLP sentiment analysis on student journal entries |
| Impact | Early warning system for students at-risk of academic breakdown |
| Repository | [Burnout-Predictor_app](https://github.com/Aditya-Choudhary-gh/Burnout-Predictor_app) |

The application features a custom maroon-black and aged gold UI aesthetic — deliberately designed to reject generic "AI tool" aesthetics and create an environment that feels calm and trustworthy for students. Features include starred contacts, data export, session switching, and a full onboarding flow.
